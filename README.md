# 🏥 МедАссистент КР — Telegram Bot

Бот для студентов медицинских вузов по клиническим рекомендациям МЗ РФ.

---

## 🚀 Деплой на Railway (бесплатно, 5 минут)

### Шаг 1 — Создайте бота в Telegram
1. Откройте Telegram, найдите **@BotFather**
2. Отправьте `/newbot`
3. Введите имя бота, например: `МедАссистент КР`
4. Введите username, например: `med_kr_bot`
5. Скопируйте **BOT_TOKEN** — он выглядит как `123456789:AAF...`

### Шаг 2 — Получите Anthropic API Key
1. Зайдите на https://console.anthropic.com
2. Создайте аккаунт и перейдите в **API Keys**
3. Нажмите **Create Key** и скопируйте ключ

### Шаг 3 — Деплой на Railway
1. Зайдите на https://railway.app и войдите через GitHub
2. Нажмите **New Project → Deploy from GitHub repo**
3. Загрузите папку `tg-med-bot` в новый GitHub репозиторий
4. В Railway перейдите в **Variables** и добавьте:
   ```
   BOT_TOKEN = ваш_токен_от_BotFather
   ANTHROPIC_API_KEY = ваш_ключ_anthropic
   ```
5. Railway автоматически запустит бота!

## 💻 Запуск локально (для теста)

```bash
# 1. Установите зависимости
npm install

# 2. Создайте файл .env
cp .env.example .env
# Откройте .env и вставьте ваши ключи

# 3. Запустите
npm start
```
